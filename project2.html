Development of a Student Enrollment Management Application

The student enrollment system, a collaborative undertaking, leverages a robust technology stack to achieve both efficiency and user-friendliness. Utilizing Laravel's framework for backend structure, Tailwind CSS for responsive styling, and Alpine.js and Livewire for dynamic front-end components, the project is implemented primarily in PHP. This synergy aims to deliver a seamless and intuitive experience for managing student enrollment processes. This is the second group project of Skill Integration course.

Tools

The Goal:

The project aimed to develop a comprehensive web application to manage degree and exchange student enrollment processes. The system provides differentiated access and functionalities for various stakeholders, including students, home institutions, international offices, landlords, coordinators, administrators, and instructors, thereby streamlining the enrollment workflow and enhancing features previously lacking in existing institutional websites.

1

The Challenge:

The upcoming influx of students unfamiliar with the website's functionalities and support channels presents a significant challenge. While the application's complexity, including its extensive database, can be daunting even in a collaborative environment, addressing this accessibility issue is a priority.

2

The Result

Significant progress has been made on the website, and final refinements are underway before its imminent launch. The platform now incorporates enhanced features, surpassing the functionality of the current school website. I Particular focused on to the buddy and application systems, as well as substantial improvements to the student system infrastructure.
